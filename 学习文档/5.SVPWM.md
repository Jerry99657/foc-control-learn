# 5. 空间矢量脉宽调制 (SVPWM) 及其极简实现

在无刷电机 (BLDC/PMSM) 的 FOC 控制中，最终计算出的 $U_q$ 和 $U_d$ 需要通过逆 Park 变换和逆 Clarke 变换转化为三相电压 $U_a, U_b, U_c$，然后再通过 PWM 波形驱动逆变器桥臂。

传统的 **SPWM (正弦脉宽调制)** 方法，是将 $U_a, U_b, U_c$ 直接加上 $\frac{U_{dc}}{2}$ 的直流偏置后，与三角载波比较产生 PWM。由于三相桥臂输出的最大相电压幅值不能超过 $\frac{U_{dc}}{2}$，这就导致了直流母线电压无法被完全利用（最高利用率只有约 86.6%）。

为了榨干逆变器的每一滴电压，**SVPWM (空间矢量脉宽调制)** 诞生了。它能让电压利用率提升 15.47%，达到 100%。

---

## 1. SVPWM 的理论基础

### 1.1 逆变器的 8 个基本状态
三相全桥逆变器有 6 个开关管，上下桥臂互补导通。因此，三相各有 2 种状态（上管开或下管开），总共有 $2^3 = 8$ 种开关状态，对应 8 个电压空间矢量。

定义开关函数 $S_a, S_b, S_c$，当上管导通时 $S=1$，下管导通时 $S=0$。
空间矢量的合成公式定义为：
$$ \vec{V} = \frac{2}{3} (U_{AN} + U_{BN} e^{j120^\circ} + U_{CN} e^{j240^\circ}) $$

以状态 `(1, 0, 0)` 为例，A 相接 $U_{dc}$，B、C 相接 GND。
此时电机中性点电位为 $\frac{U_{dc}}{3}$。
三相绕组的相电压为：
$U_{AN} = \frac{2}{3}U_{dc}$
$U_{BN} = -\frac{1}{3}U_{dc}$
$U_{CN} = -\frac{1}{3}U_{dc}$

代入矢量合成公式，得出基本矢量 $\vec{V_1}$ 的幅值为：
$$ |\vec{V_1}| = \frac{2}{3} U_{dc} $$
同理，状态 `000` 和 `111` 合成矢量的幅值为 $0$，被称为**零矢量**。其余 6 个**非零基本矢量**的幅值均为 $\frac{2}{3}U_{dc}$，它们在空间中形成一个正六边形。

### 1.2 伏秒平衡原则与时间分配
SVPWM 的核心思想是：**利用相邻的两个非零矢量和两个零矢量，在一个 PWM 周期 $T_s$ 内合成任意期望的电压矢量 $\vec{V}_{ref}$。**

假设目标矢量 $\vec{V}_{ref}$ 落在第一扇区（由 $\vec{V_1}$ 和 $\vec{V_2}$ 夹成），根据**伏秒平衡原则**：
$$ \vec{V}_{ref} \cdot T_s = \vec{V_1} \cdot T_1 + \vec{V_2} \cdot T_2 + \vec{V_0} \cdot T_0 $$

因为基本矢量幅值为 $\frac{2}{3}U_{dc}$，通过对该复数方程在 $\alpha$ 轴和 $\beta$ 轴展开，可以解出导通时间 $T_1$ 和 $T_2$。剩下的时间分配给零矢量：
$$ T_0 = T_7 = \frac{T_s - T_1 - T_2}{2} $$

为了减小开关损耗和降低谐波，在 STM32 的 **中心对齐 PWM 模式** (Center-Aligned Mode) 中，我们会采用 $T_0 - T_1 - T_2 - T_7 - T_2 - T_1 - T_0$ 这种**7 段式对称发波**顺序。

---

## 2. 传统扇区法的缺点

虽然上述理论非常严谨，但在单片机 C 语言实现时却极度繁琐：
1. 需要计算逆 Clarke 变换或查表来判断当前目标矢量落在哪个**扇区 (Sector 1~6)**。
2. 对于不同的扇区，需要使用不同的数学公式去计算 $T_1$ 和 $T_2$。
3. 需要根据扇区决定时间 $T_1, T_2, T_0$ 该分配给 $CCR1, CCR2, CCR3$ 中的哪一个。

这些层层嵌套的 `if-else` 分支极大地消耗了 CPU 的时钟周期，并且非常容易写出 Bug。

## 3. Min-Max 零序电压注入法：代码的艺术

在现代 FOC 库（包括我们的 `JerryFOC` 本工程）中，我们采用了在数学上与传统 7 段式 SVPWM **完全等效**、但代码却极其优雅的**马鞍波注入法 (Zero-Sequence Injection, 简称 Min-Max 法)**。

### 3.1 数学原理
在线性代数中，对于没有中性线引出的三相电机，在三相相电压 $U_a, U_b, U_c$ 上同时加上一个相等的“共模电压 $U_{com}$”，**电机的线电压完全不变**，因此电机的转矩和电流也毫无察觉。

$$ U_{ab} = (U_a + U_{com}) - (U_b + U_{com}) = U_a - U_b $$

我们如何利用这个“共模电压”来实现 SVPWM 呢？
为了让 PWM 占空比在 $[0, 100\%]$ 内对称，并且最大限度利用母线电压，我们让三相电压中**最大的一相**和**最小的一相**的平均值，正好等于直流母线电压的一半 $\frac{U_{dc}}{2}$。

也就是求解方程：
$$ \frac{(U_{max} + U_{com}) + (U_{min} + U_{com})}{2} = \frac{U_{dc}}{2} $$
得出神奇的零序注入公式：
$$ U_{com} = \frac{U_{dc}}{2} - \frac{U_{max} + U_{min}}{2} $$

将这个 $U_{com}$ 加回纯逆 Clarke 变换出来的正弦波上，正弦波就会被扭曲成经典的“**马鞍波**”，这正是 SVPWM 相对于中性点特有的波形！

### 3.2 JerryFOC.c 的具体实现
在我们的工程中，TIM1 被配置为了 **中心对齐模式 3 (TIM_COUNTERMODE_CENTERALIGNED3)**，这天然为产生对称波形做好了硬件准备。

以下是我们用短短几行代码彻底替代复杂扇区判断的全部逻辑：

```c
// 1. 纯逆 Clarke 变换：得到以 0 为中心的纯正弦三相电压
float Ua = Ualpha;
float Ub = -0.5f * Ualpha + (sqrtf(3.0f)/2.0f) * Ubeta;
float Uc = -0.5f * Ualpha - (sqrtf(3.0f)/2.0f) * Ubeta;

// 2. SVPWM 核心：找出三相电压中的最大值和最小值
float Umax = Ua > Ub ? (Ua > Uc ? Ua : Uc) : (Ub > Uc ? Ub : Uc);
float Umin = Ua < Ub ? (Ua < Uc ? Ua : Uc) : (Ub < Uc ? Ub : Uc);

// 3. 计算零序电压 Ucom (马鞍波中心偏移量)
float Ucom = (voltage_power_supply / 2.0f) - (Umax + Umin) / 2.0f;

// 4. 将零序电压注入，完成 SVPWM 调制！
Ua += Ucom;
Ub += Ucom;
Uc += Ucom;
```

### 3.3 结论
采用 **Min-Max 法**后，我们不仅免除了所有的三角函数查表和繁琐的扇区判断，节省了至少 30% 的算力，而且通过 TIM1 的中心对齐模式发出的 PWM 波，其实际驱动电机的效果与死磕教科书手算的 $T_1, T_2$ **完全一致**。这就是工程实现上的极致优雅。
